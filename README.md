# Герундий vs инфинитив — гайд по английскому в стиле Shyngys

-ing или to. Предлоги-триггеры, слова-боссы, ловушка Stop, Go + активность. Интерактивный тренажёр-сборщик фраз.

Одностраничник в фирменном рубленом стиле (короткий личный крючок, метафоры вместо терминов,
контрастные примеры). Эстетика Apple, mobile-first, крупная типографика, плавные scroll-анимации.

## Что внутри
- Текст в стиле коротких рубленых фраз с личным анекдотом
- 🔊 Озвучка примеров (Web Speech API, en-US)
- Анимации: hero с покадровым reveal, floating-блобы, анимированный SVG-визуал, scroll-reveal, прогресс-бар
- Интерактивный тренажёр с мгновенным объяснением
- FAQ-аккордеон

## Запуск локально
```bash
python3 -m http.server 8000
```
Один файл `index.html`, всё внутри (кроме шрифта Inter).

## Деплой
GitHub Pages, ветка `main`, путь `/`.
